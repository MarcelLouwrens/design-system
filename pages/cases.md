<h1>Special Cases</h1>

<p>To encourage creativity, and to accommodate exceptions, special emphasis or unique UI scenarios, changes to the suggested components in this guide can be made.</p>

<p>These special cases are bound to the following rules:</p>

<ol>
	<li>The grid needs to be adhered to.</li>
	<li>The colour palette needs to be adhered to.</li>
	<li>The font cannot be changed, except in cases where Monospace fonts are required.</li>
</ol>
